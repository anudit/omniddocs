Multi-hop Proxy Re-encryption, also known as Transform Encryption, is a cryptographic technique that allows data encrypted with one person (Alice's) public key, to be transformed, so that it can be decrypted using another person (Bob's) private key, without revealing either person's private key. 
This transformation relies on a special key calculated from the first person's private key and the second person's public key.

## @omnid/pre
A proxy re-encryption implementation using the secp256k1 curve.
