Function Cloud is a fast proof execution environment supporting multiple backends. This is particularly useful for edge/IoT devices with limited computing capabilities.

<Callout type="info">
We will soon be adding support for private/MPC powered proofs that hide inputs securely too.
</Callout>

FC supports the following proving backends,
1. Circom
2. Halo2
3. Noir
4. Plonky3
5. Risc0
6. SP1

https://fc.omnid.io
